# Listing Admin Web (Next.js)

Minimal admin dashboard for managing categories, products and product images.

## Setup
1) Create `.env.local`:
```
NEXT_PUBLIC_API_BASE_URL=http://127.0.0.1:5000
```

2) Install & run:
```cmd
npm install
npm run dev
```

Open http://localhost:3000

## Notes
- No auth in this scaffold. Add auth before production.
- Orders page is a placeholder until we add orders endpoints to the API.
